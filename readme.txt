Lightcase Lightbox Plugin for WordPress
====================================

This plugin will implement the lightcase lightbox into WordPress

Installation Instructions:

1. Activate the Plugin

The Plugin will automatically be active on all gallerys and on all posts and pages, automatically grouping the images together.

Custom Styling for Images:

If you wish to customize the styling of the images, copy the custom.css in the css directory, over to your theme to a file named lightcase_custom.css, the plugin will take your custom css file over the default one.
